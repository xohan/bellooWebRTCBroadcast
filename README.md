# WebRTC Video Broadcast

Sends video from a device (broadcast.html) to a viewer (index.html) over WebRTC.

Uses WebRTC's getUserMedia(), RTCPeerConnection(), `<video>`, `<canvas>`, Node.js and Socket.io
